How to publish changes
Everything here is local — nothing auto-saves to GitHub. Follow these steps:
Step 1 — Export

Click Export data.js in the top bar. A file called data.js is downloaded to your Downloads folder.

Step 2 — Replace the file

Move the downloaded data.js to imagemedia-redesign/assets/data.js, replacing the old one.

Step 3 — Push to GitHub

Open Win + R → cmd, paste this command and press Enter:

cd /d "C:\Users\user\Documents\Claude\Projects\Website\imagemedia-redesign" && git add assets/data.js && git commit -m "Content update" && git push origin main
Changing page text or HTML

For things like page headlines that are in the HTML files (not data.js), ask Claude to edit the file — just say what you want changed and Claude will update and push it.
